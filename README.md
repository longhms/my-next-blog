# ⚙️ NextJS Pitchside Press Blog

Đây là một dự án blog được xây dựng nhằm thực hành và áp dụng các tính năng mới nhất của **Next.js (v15+)**.  
Mục tiêu chính là tối ưu **hiệu năng**, cải thiện **trải nghiệm người dùng** và tuân theo **best practices** trong phát triển web hiện đại.

## ✨ Điểm nổi bật

- ⚡ **Tối ưu tốc độ tải trang** với SSG và ISR  
- 📱 **Thiết kế responsive** hoạt động tốt trên mọi thiết bị  
- 🔍 **Tìm kiếm & lọc bài viết** trực tiếp qua URL  
- 📝 **Hỗ trợ Markdown** cho việc viết nội dung  
- 🛡️ **Kiểm tra dữ liệu form** bằng Zod và React Hook Form  
- 🌙 **Sẵn sàng cho Dark Mode** với cấu trúc dễ mở rộng  

---

## 🛠️ Tech Stack

| Công nghệ        | Mục đích sử dụng |
|----------------|----------------|
| Next.js 15+    | Framework chính (App Router, Server Components) |
| TypeScript     | Định nghĩa kiểu dữ liệu |
| Tailwind CSS   | Xây dựng giao diện |
| Zod            | Validate dữ liệu |
| React Hook Form| Xử lý form |
| Lucide React   | Icon |
| Vercel         | Triển khai ứng dụng |

---

## 🏗️ Project Structure
├── app/ # Routing, layout, API (Next.js App Router)<br>
├── components/ # UI components tái sử dụng<br>
├── data/ # Dữ liệu bài viết (JSON)<br>
├── lib/ # Hàm tiện ích & validation<br>
├── public/ # Static assets<br>
├── types/ # Type definitions<br>
└── next.config.ts # Cấu hình Next.js<br>

## 🚀 Run locally

### 1. Clone repository

Bash
git clone https://github.com/longhms/my-next-blog.git

### 2. Install dependencies

Bash
npm install

### 3. Start development server

Bash
npm run dev
Truy cập http://localhost:3000 để xem kết quả.
